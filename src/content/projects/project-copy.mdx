## 프로젝트 개요

- 키워드 기반 투자 인사이트 제공 서비스
- 뉴스 감성 분석 + 키워드 그래프 제공

## 기술 스택

- Front: Next.js, Tailwind CSS
- Server: FastAPI
- Infra: AWS (ECS, RDS, S3)

## 주요 기능

1. 키워드 연관도 분석
2. 감성 기반 색상 시각화
3. 트리 그래프 기반 키워드 탐색
