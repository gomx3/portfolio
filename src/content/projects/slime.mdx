### 기여 역할

    - Three.js & R3F 기반의 3D 렌더링 환경 구축: Scene, Camera, Light 등 3D 필수 요소를 React 컴포넌트 방식으로 모듈화하여 설계
    - 실시간 인터랙션 로직 구현: 사용자의 마우스 움직임과 컨트롤러 입력값을 3D 객체의 Mesh 형태 변형에 실시간 반영
    - 상태 관리 최적화: Jotai를 활용해 컨트롤러의 UI 상태(DOM)와 Canvas 내부 3D 객체 상태(WebGL) 간의 효율적인 동기화

### 배운 점

    - 2D 화면이 아닌 3차원 공간에서의 좌표계, 조명, 재질(Material)이 어떻게 웹 브라우저에 렌더링 되는지 깊이 있게 학습했습니다.
    - 일반적인 DOM 조작을 넘어, 웹 기술로 구현할 수 있는 시각적 경험의 한계를 넓혔습니다. 새로운 기술 스택(Three.js)을 스스로 학습하고 적용하는 과정을 통해 "새로운 기술을 학습하고 러닝 커브를 두려워하지 않는 개발자"로서의 자신감을 얻었습니다.
